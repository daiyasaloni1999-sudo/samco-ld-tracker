Deploy to Netlify — Step by step
Estimated time: 20 minutes. Result: permanent URL like samco-ld.netlify.app
1
Download this file
Save index.html and netlify.toml from your advisor.
2
GitHub — create repository
github.com → New repository → name: samco-ld-tracker → Public → Create.
3
Upload both files
Inside repo → Add file → Upload files → drag both files → Commit changes.
4
Netlify — connect & deploy
netlify.com → Sign up with GitHub → Add new site → Import existing project → GitHub → select samco-ld-tracker → Deploy site.
5
Rename your site
Site settings → Change site name → type samco-ld → your URL: samco-ld.netlify.app
6
To update later
Replace index.html in GitHub → Netlify auto-redeploys in 60 seconds. Your data is safe in browser storage.
