Deploy to Netlify (Option 3) — Step by step
Estimated time: 20 minutes. Result: a permanent URL like samco-ld.netlify.app accessible from any device.
1
Download this file
Click the download button your advisor gave you to save index.html to your computer. Also download netlify.toml.
2
Create a GitHub account & repository
Go to github.com → Sign up → New repository → Name it samco-ld-tracker → Public → Create repository.
3
Upload the file to GitHub
Inside your new repo → click Add file → Upload files → drag index.html and netlify.toml → click Commit changes.
4
Connect Netlify
Go to netlify.com → Sign up with GitHub → Add new site → Import an existing project → GitHub → select samco-ld-tracker → Deploy site. Done in 60 seconds.
5
Your live URL
Netlify gives you a URL like random-name.netlify.app. Go to Site settings → Change site name → rename to samco-ld → your URL becomes samco-ld.netlify.app. Share this with no one — it's your personal tracker.
6
Connect your Google Sheets
Open your live URL → go to Settings tab → paste your two published CSV URLs → Save → click Sync. From now on, every new form submission auto-appears in your tracker.
Any time you want to update the app (new features), just replace index.html in GitHub. Netlify auto-redeploys in under 60 seconds.

How to get your Google Sheet published CSV URL (one-time setup)

  1. Your Google Forms already have linked sheets — open the response sheet
  2. Go to File → Share → Publish to web
  3. Select the Form Responses sheet tab → change format to CSV → click Publish
  4. Copy the URL — it looks like: https://docs.google.com/spreadsheets/d/e/…/pub?gid=…&output=csv
  5. Paste it below and click Save → then Test fetch
